# Content API Turbo – Textos para atendimento, resumos e e-mails.

![Python](https://img.shields.io/badge/python-3.11-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-1.38-red)
![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4o--mini-green)

**Projeto criado em novembro/2025 para testes com a API da OpenAI**

## Funcionalidades
- Respostas empáticas de atendimento
- Resumos de cardápio com emojis
- Promoções com urgência
- E-mails profissionais para parceiros

Tudo com GPT-4o-mini + Streamlit. Zero configuração além da API Key.

```

cd content-api-turbo
pip install -r requirements.txt
streamlit run app.py
```
